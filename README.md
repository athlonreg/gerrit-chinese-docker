# gerrit-chinese-docker
Gerrit Docker For Chinese Language.
# Pull
```bash
# docker pull canvas1996/gerrit-2.16.3-chinese:latest
```

# Run
```bash
# docker run -ti -p 8080:8080 -p 29418:29418 canvas1996/gerrit-2.16.3-chinese:latest
```

# Port
- Web -- 8080
- SSH -- 29418
